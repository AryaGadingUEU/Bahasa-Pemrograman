Kesimpulannya adalah:
Dari segi looping, perbedaan antara continue dan break adalah
Kalau kodingan kita menggunakan continue, maka perulangan akan diulang terus sampai sampai akhir dan variabel â€œhâ€ akan dilangkahi / dilewati. 
Sedangkan kalau kodingan kita menggunakan â€œbreakâ€, maka perulangan akan dihentikan seketika perulangannya telah mencapi â€œhâ€ dan tidak diteruskan lagi perulangannya sejak â€œhâ€.
